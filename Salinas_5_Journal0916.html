describe what each of the following does:

  1. ol,li - creates a list that are numbered
  2. ul- creates a list that are bulleted
  3. tr- defines a row in a table
  4. td-stuff inside of a table
  5. a- makes a link, usually that goes to another page
  6. aside- puts the content aside
  7. footer- defines a footer for a document or a section
  8. margin-left,margin-right-puts the stuff inside to the left/right hand side of the screen
  9. width- sets the width of a object, piture, or text
  10. height- sets the height of a object,picture,or text
  11. background-color- sets the background color to whatever color is set
  12. color- sets the color for whatever is inside the tags
  13. a:hover- when mouse is ontop of a text or picture, it will change color/ do something
  14. text-decoration- sets a certain attribute to the text
